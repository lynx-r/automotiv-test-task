# Запуск проекта в Docker

```bash
docker-compose up -d
```

Далее, нужно подождать пока выполнится сид в базу данных.

Frontend http://localhost:8080

Backend http://localhost:3000

# Оптимизация

В качестве оптимизации было сделано следующее:

1. Индекс по name
2. Кеширование всех эндпоинтов

После добавления индексов существенной производительности не обнаружил.

После добавления кеша время обработки 100 запросов уменьшилось
с ~3500 мс до ~850 мс при нулевой задержке между запросами и ~14000 мс
против ~11000 мс при задержке между запросами в 100 мс.
